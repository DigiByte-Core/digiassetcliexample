# digiassetcliexample
CLI example for digiassets

## Installation
  $ npm install -g digiassetexample

## Requirements
  DigiByte wallet or another way to create digibyte addresses and sign transactions

## Arguments
```
-u DigiByte RPC User
-p DigiBytye RPC Password
-o DigiByyte RPC Port
-v Verbose
```

## Running
  $ digiassetexample
